## こうかとんクリッカー

### 実行環境

- Python >= 3.10
- pygame >= 2.5

### ゲームジャンル

クリッカーゲーム

### ゲームの概要

こうかとんをクリックし、**金**を増やしていくゲーム。

貯めたスコアで右のアイテムを購入。スコアがすごい勢いで貯まるぞ！

## ゲームの実装

### 共通基本機能

- 背景画像
- 文字列の表示
- キー入力
- オブジェクト思考

### 追加機能

- 画面クリック
